# Architecture Decision Records (ADR)

Documented technical decisions and their rationale.

## Format

Each ADR follows the template:

```markdown
# ADR-NNN: Title

## Status
Proposed | Accepted | Deprecated

## Context
What is the issue?

## Decision
What was decided?

## Consequences
What are the trade-offs?
```
